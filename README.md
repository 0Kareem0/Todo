# ✅ Todo – Minimalist Task Manager

> "Get things done. One task at a time."

---

## Overview

**Todo** is a **clean, fast, and distraction-free** to-do list app built with **vanilla JavaScript, HTML, and CSS**.  
No frameworks. No bloat. Just **pure, semantic code** that works instantly.

Features:
- Add, edit, delete tasks
- Mark as complete with smooth animations
- Persist data in `localStorage`
- Fully responsive & mobile-ready
- Dark/light mode toggle
- Zero external dependencies

> *Frontend fundamentals done right — perfect for learning DOM, events, and state management.*
